# AI Swing Trader

AI Swing Trader is a Python-based AI-assisted swing trading decision support system for the Indian stock market.

The application is designed to analyze Nifty 500 stocks, identify high-probability swing trading opportunities, and generate explainable Buy, Hold, and Sell recommendations.

## Planned Features

* Automatic Nifty 500 data collection
* Market analysis
* Sector analysis
* Technical analysis
* AI opportunity scoring
* Excel dashboard
* Portfolio tracking
* Historical backtesting
* Slack and Email alerts
* AI-generated explanations

Currently under active development.
